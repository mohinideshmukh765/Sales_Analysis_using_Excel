# B&J Biscuit Business Analysis Dashboard

![B&J Biscuit Dashboard Preview](Sales Dashboard.png)

A comprehensive, interactive business intelligence tool built in **Excel** to analyze revenue, profitability, and customer demographics for B&J Biscuit. This project transforms raw sales data into actionable insights to optimize marketing and sales strategies.

## 📊 Project Overview
The primary goal of this dashboard is to provide a high-level view of business health while allowing stakeholders to drill down into specific performance metrics.

### Key Visualizations & Insights
* **Revenue Distribution:** Analyzed by product price categories (High vs. Low), age groups, gender, and payment method.
* **Profitability Analysis:** Identification of the most profitable brand, location, customer, and salesperson.
* **Customer Insights:** Ranking of top 5 customers by revenue contribution and total customer acquisition tracking.
* **Geographic Performance:** Revenue share across key locations like San Antonio, Houston, and Dallas.
* **Sales Metrics:** Real-time tracking of Quantity Sold, Total Revenue, Cost, and overall Profit Margin.

## 🛠️ Features
* **Interactive Slicers:** Filter all charts and KPIs by **Location**, **Payment Method**, and **Age Group**.
* **Visual Toggles:** Support for switching views between **Absolute Values** and **Percentages** for comparative analysis.
* **Clear Navigation:** Dedicated "Clear Slicers" functionality to reset the dashboard view instantly.

## 🚀 Technical Implementation
* **Tool:** Microsoft Excel (B&J Buscuit Practice Dataset `.xlsm`).
* **Architecture:** Utilized a **Monolithic** structure to showcase specific skills, ensuring all logic and visual layers are contained within a single file.
* **UI/UX:** Designed a clean, professional interface with distinct KPI cards and coordinated color schemes.

## 📁 Repository Structure
* `B&J Buscuit Practice Dataset.xlsm`: The core macro-enabled Excel file.
* `Sales Dashboard`: A high-resolution preview of the final dashboard interface.
* `README.md`: Project documentation.
